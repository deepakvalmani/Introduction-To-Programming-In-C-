1. **Program Structure**: main() function, includes, namespaces
2. **Input/Output**: cout, cin, endl, stream operators

4. **Operators**: Arithmetic (+, -, *, /, %), assignment (=), bitwise (>>)
5. **Control Structures**: if statements, while loops, for loops, do-while loops
